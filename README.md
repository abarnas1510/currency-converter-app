# Currency Converter
## Project Description
The Currency converter is a Python-based application that converts an amount from one currency to another using predefined exchange rates.
## Technologies Used
-Python
-Tkinter(for GUI)
-Real-time API
## Input
-Amount
-From Currency
-To Currency
## Output
-Converted currency amount displayed to the user
